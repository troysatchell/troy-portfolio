# Troy Satchell — Portfolio

Personal portfolio site for Troy Satchell (Software Engineer / Founder, [Clavira](https://clavira.com)).

Single-page static site: `index.html` + `screenshots/`. Tailwind + Phosphor icons via CDN, no build step.

## Run locally

Any static server works. From this directory:

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Deploy

Push to `main`. Enable GitHub Pages in the repo settings (Source: `Deploy from a branch`, Branch: `main /root`) to publish at `https://troysaved.github.io/troy-portfolio`.
